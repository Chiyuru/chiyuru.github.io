近期同人产出/练习合集

发表于 2025-08-28 更新于 2026-01-15

不知道为什么在一个极度焦躁的时刻突然重拾了多年前的爱好，但比起记录画了什么，主要想写的是绘画过程中的情绪和考虑。

为了让这一篇在博客里往下沉一沉，将写作日期往前改了四个月。

其实我的绘画基础是零，没有正式学过任何技法，但追溯起来的话从小学看名柯的时候就经常涂涂画画，初中搞瓶邪的时候有稍微画过一点原创，应该也没获得什么热度，已经找不到了。后来因为各种原因（学业上变忙、初升高的时候我的心理健康出现了很大的问题、如今绘圈的风气也变化了）没有再继续。

2025 年年底申请季中间我还有好几门专业课要考试，在这个时间点正好遇到鬼灭之刃无限城 1 上映，本来只是当商业大片随便看看，但我这该死的 xp 从来没变过就是一眼爱上了义勇，简单刷了一些剧情之后磕到了义炭 cp。也许只是因为这段时间压力太大加上心情不好，虽说要做的事情很多但整体的状态却是一味地焦躁，时间上反而有很多空闲，也可能是因为义炭实在太有故人（瓶邪）之姿，我就这么又开始搞同人了。

一开始还是单纯地吃饭，同人文看了一些感觉索然无味，后面又吃了大量 goshi, Ruint 和海崎老师的同人本，尤其 goshi 的几个传世经典 R18 本真的给我看压抑了（，虽然话是这么说实际上这一对我还是更喜欢清水。转机发生在 12.26 那天，又一次在和家人的电话中接受到（我认为无理的）指责和恨意之后，心里再一次涌起了“我到底是因为什么现在还活着”的想法，想刷一会同人冷静一下，在这时邂逅了『雨の日、君に会いにいく』（R18 注意）。本子本身没有非常戳到我，然而 P1 义勇在海边的背影让我一下子想起了月初在七里滨看到的海滩，从藤泽直到镰仓连绵不绝涌动不止的海，P3 炭说自己是第一次见到海，描绘的景色和我初次在江之岛看到的海和富士山那么相似。所以我也见过令人感到幸福的景色，也许也能有美丽的事物从我的手中诞生，那么我可能也就还可以继续活着，我最初就是抱着这样的心情摊开了画纸。

目前没有搞原创的动机和心力，只是觉得画得开心、能看到自己手里也会诞生漂亮的东西就好。谨慎起见，可以发的以原作画面重绘为主。感觉在这里 CP 向角色向也没必要分开，还是按日期记（

如上所述，拿着一支自动铅一块橡皮喊着什么勇气啊羁绊啊地就开始临了。是 『雨の日、君に会いにいく』 的 P1。

【假装有图】

试了手头的几种纸之后感觉还是北大的作业纸手感最好，虽然胸口仍然翻腾着无法平息的愤怒和疑问，但起型的过程很顺利，勾出义勇的轮廓和地平线之后先没怎么管海浪，就开始细化和给人物部分涂色了。上色的过程中感觉心情逐渐平复了下来，像在玩初中时候流行的秘密花园填色本，由于我只有一支自动铅笔，在这个过程中倒是学会了用铅芯的不同切面和角度画出不同的黑白灰，虽然并没有理解为什么黑白灰是这样画的，很困难但反而让我感觉平静。

其实这个图就应该在画完人物的时候停下的，我临的时候完全没理解海浪要怎么表现，只能试着一比一复刻，虽然复刻得很丑但是画完就突然懂了。最后也就到此为止没有好好地去修细节，包括沙滩就直接没画了，但没有画雨是因为想起之前说的后期画风上义勇瞳色的变化，瞳孔周围的浅色部分越画越亮，整体也没有了雪地初见那种高级灰的色调，倒是更符合水柱之名像晴空下大海的颜色，就希望能够保留这样的印象。之前也和朋友开玩笑说不知道水柱有没有看过海，没看过的话可以跟调查兵团一起去（，又想起看海名场面里阿尔敏的瞳色，倒是和无限城里义勇亮亮的眼睛有点相似。

然而起型起的太大了，导致这个头发真的涂了很久，富贵鱼鱼的头发真的有好多片，尤其老师的画风是把马尾画成像一大把柚子叶一样的形态，很可爱但手工上色真的好累。涂色只能说尽可能均匀了但 HB 自动铅笔的极限摆在这里，实在无法画出很明显的对比色。在这张上面学到了天使环高光，虽然目前工具所限只能手动抠高光的轮廓然后把其他地方涂黑。其实这张的手和眼神都不难刻画，某种程度上来说也是画大图的优点。

按照 Gemini 的建议购入了 4B 铅笔和蜻蜓的 mono zero，贵我一跳，但估计够用了。

我馋这个富贵鱼鱼的腰很久了，这个把羽织飘起来的画法只能说是他先 going 我（，临了。然而起型并不顺利，这个手很难画，两只手都是抓握的姿势需要把手指画得粗短一点，审美和透视在打架，导致拔刀的右手手指因为画得太细看起来很怪也没有力量感，实际上想突出手指的纤长应该在骨节和指甲的地方强调，手腕旋转的关系也没画好，应该拿自己的手比划一下的。左手稍微容易一点，无名指也还是画短了。其实眼睛也有点细节不够，但铅笔太粗了起型又起的太小导致没办法细化，我虽然明白这张的高光怎么画但也不好抠，只能等铅笔橡皮到货。

叠甲：我平时主要打反曲弓，和弓的形制参考了网图，动作是参考了我打弓时候的照片。没有系统学过美术，只临摹过一些喜欢的作品，光影等方面不专业致歉。

洗澡的时候突然在想你们鬼杀队为啥全都是近战啊，但凡有一支弓箭队也不至于平时杀个普通的鬼也那么费劲，况且也有像悲鸣屿这种不用剑的战士，但确实没有用弓箭的角色。

当然鬼灭之刃是一部不能深究的作品，真正让我想到画这幅画的是想起义勇打上三开斑纹时候的独白，头一次尝到了死斗的滋味，在和强者的对决中短时间内感官和剑术被磨砺到极致，这种向内求的精神让我想起练反曲弓的时候一度觉得瞄准靶心和射击结果最重要，后来在逐渐的修炼中明白了动作一致性的重要性，如果身体的各个部位正确地参与发力，瞄准的时候就不会有那么多杂念，开弓看清弦影弓把的位置再对一下瞄准具撒放即可，换言之，与其求外在的目标不如专注身体的调整、信心的建立。这让我觉得义勇似乎也是一个很适合弓箭的角色，当然鬼灭之刃的时代没有现代竞反，主包也没有打过传统弓更别提见都没见过和弓，紧急扒了一点参考就画了。当然有很多问题，光影透视基本上在凭感觉乱画，因为用自己的照片做参考的原因好像小臂画的太细了（笑

跨年在画富贵鱼鱼，有点抽象。

详见 link

本来想着滑铲个 Q 版应该赶得上但还是晚了半个小时，可恶！

参考柱训练篇官图画的 Q 版，加入了一点点新年元素，多年前看卫宫家今天的饭的时候就了解到新年吃荞麦面的习俗，正好义炭有这个桥段，不妨在新年之际开展一场堂堂正正的对决（x）

也是因为没来得及滑铲的原因，跨年的时候还在狂画义勇的头发，就这样画着 cp 迎来了新年。我是年底上映无限城的时候入坑了鬼灭，一眼就爱上了义勇和义炭 cp，正好最近压力太大搞同人有一种精神支柱的感觉，从零基础开始练习画了不少图。也许这会是非常短暂的喜爱，等压力没那么大了就不会再高强度参与 cp 建设，但无论如何此时的心情非常真诚，新年也确凿地在画画中度过了，搞 cp 教会我最大的道理大概就是一定要抓住短暂的有力量去爱的机会，要说的话就是此刻了。

感谢阅读，谨祝新年快乐！

说真的这篇在小红书的流量非常烂，但在 Lofter 数据不错，有点难懂。成图其实我自己也不怎么满意，首先是赶时间太粗糙，上色有 bug 但我甚至都不想拿出来再修，再就是我自己其实也不喜欢这种形式的半精致 Q 版，以后估计也不会继续练或者画。

我流狂草低脂四格小漫画，ooc 致歉

梗源：致敬传奇音游人富冈义勇 http://xhslink.com/o/2aBFVMnI3fa 复制后打开【小红书】查看笔记!

感谢炭几喽送来的通透世界，不然大家还不知道凪是鱼手动挥剑挡出来的，那田蜘蛛山的时候看鱼一动不动原地挥出一个凪，实际上原来是在大家看不清的时候匆匆忙忙连滚带爬的音游 all perfect 吗，致敬传奇音游人富冈义勇！

主包很久没玩音游了，第一反应了解的最难谱面仍然是 #arcaea 的 #GrieviousLady，至于为什么不是理论值，只能说还得练啊鱼（逃

说真的这篇在小红书的流量非常好，比之前的整图还要好，但在 Lofter 烂了，我完全搞不懂这两个平台。画完这个就不打算继续原创新图了，尽管我现在脑子里有八百个 idea 想画，但这样下去无甚意义，我还是想回去继续临摹也好学习素描也好，总之做些成图更漂亮且有意义的事情。

看到这个数据的时候说完全不觉得高兴肯定是假的，但实际上这图也不咋样主要是梗太有意思了，大概也是我第一次用水笔画狂草小漫画，P1 明显还没适应手感，面部画得完全不像鱼鱼，但这个图我是打算攒起来印无料的所以之后还得改一下。如果是板绘的话前三张可以在同一个图上面改就不用画出三张感觉完全不同的图，只可惜我极其讨厌手写笔的手感，平时用它写作业做 cheatsheet 之类的都是迫不得已，画画的时候还是想古法纸绘，这都是没有办法的事。

不知怎么的我想起了我的第一篇 paper，到现在我已经发表/投稿了两篇顶会文章，然而我从未在这个过程中感觉到创作的快乐，这再正常不过，都是为了毕业和申请。哪怕我们已经在努力向前沿靠近，我也知道这两篇文章实际上“习作”的意义大于“研究”，第一篇文章的 code repository 开完会了也无人问津，我都怀疑我的有生之年能否看到有谁去用这个东西，做科研截至目前也是一种工作而非创作，因而我的热心程度、从中获得的意义感尚不如发现这个低脂四格小漫画被三千多人看过。不过，虽然我在科研习作里暂时还没找到什么创作的意义感，但我遇到的每一个老师和合作者都是非常非常好的人，能建立这样的连结，让我十一月份 rebuttal 的时候再次听到老板的声音和问候都感到非常高兴。

然而，为什么要从中获取意义？回想我因为专业错误而如同噩梦的本科五年，在学业科研中最快乐的时刻还是中间狠狠上数学课的两年，学会了很多优美的东西（尽管后来也忘了），考了不错的成绩，这和临摹很像，都是重复他人完成的精妙的作品，掌握技法就能达到好的结果，我很熟悉，也很喜欢，但人在本科阶段没有工资没有 offer，科研上创作的壁垒远高于绘画，成就感也远远不如。学数学课的那两年我也从来不说什么意义不意义的，不会去考虑这个事情。近两年我的家人身体上总是有些毛病，性格偏激尖锐更甚从前，尽管从前我也经常收到“我这辈子最后悔的事就是生下了你”之类的指责，频率尚在我能够理解这话是在向我抱怨早已结束的所托非人的婚姻的痛苦和身体的不适，尚在我流过泪就能忽略的范围里。直到我暑研结束前最忙碌的数天持续地被电话骚扰说“快去死吧，死在外面不要回来”，直到圣诞节附近的几天我一直听到“我最恨的人就是你，我不可能放过你”，我实在无法再正常地工作，甚至不能去听朋友的 talk，于是逃进了临摹、同人志和低脂小漫画里，直至现在。

我之前和亲友开玩笑说樱花妹不吃日黑是因为这玩意 neta 源氏兄弟还 ooc 了实在吃不下，我不吃是因为实在不能再承受恨海情天，我的生活里的恨比这更沉重，而情天却不多。相比之下义炭是多么健康的相互托举和保护的感情，我真的非常喜爱，或者说憧憬，建设 cp 对我来说就是搞砸了一切的自己手中诞生出美丽的事物，将我眼睛里曾经倒映的令人幸福的景色再反哺回去。荒谬当道，爱拯救之，新年的愿景除了有个可以接受的学上之外，大概就是这样了。

2026.01.11-01.14

最近开始尝试搓短打，之前一直不是很敢产出文字，现在觉得可能多半是因为一直看巨人和 EVA 这种已经完善自洽的作品所以没有很好的切入点去写，EVA 我不吃 cp，巨人的话只能说谏山创实在太强了我无法插手，把人体练到比谏山创强之后倒是还有希望画点画。相比之下闺蜜の刀这种无法推敲的千疮百孔的非严肃作品就很适合练手，可以先从写一个场景的短打开始，我开始搓了才意识到文手被平台限流的严重程度。而且我仍然捉摸不透这个平台的推流，写的原作向低脂内容我自己都不敢回头去看结果比我特别喜欢的几篇女神的产出还高，另两个写完很喜欢的 paro 反而是没人看。

不过说实话我觉得 paro 还是很好写的，就比较类似于 A+B 式科研，想写什么样的 paro 都先自己脑一脑大概有哪些可以用的梗，在这个过程中很容易找到想写的饺子醋，把角色放进去之后再加点有作品特色的 feature，给自己写美了之后再打磨下语言，饺子就端上来了。

但也有弊端，写不出来的时候是真的写不出来，画不出来的时候好歹还可以不带脑子直接上点练习。
