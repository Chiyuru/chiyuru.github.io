4月19日

开始跑 simulation，上次看到去年暑研姐写的代码眼前一黑，统计人不写 R 写 Python 就像，后面忘了（，今天先重新配了环境才有信心开始看代码（。我已经两年半（警觉）没有写过 Python，打开 workspace 看到大一的暑假写的代码，感觉纯是学数学学废了（

话又说回来，先把 latent variable 估出来然后 diffusion，然后说效果比其他方法直接对 graph 转成的 0-1 vector 做 diffusion 要好，那不然呢，这不欺负人呢吗（。我觉得把 universal latent space model 塞进 hypergraph 这个工作确实做得很好，后面这些包括我马上要干的活都真有点神秘特技了。

但他们终于把 code 开给我之后还是没把 supplementary material 给我看，唉但愿写证明的时候能看吧，总不能还不让照抄得自己写吧，不是（（。也许如果早点开始的话能赶上 NIPS 的，现在除非天天爆肝否则肯定来不及了，我还有 Jing 的 project，祈祷双开不要爆炸就已经是奇迹。Ji 这边真要写两篇的话，我确实不希望它们都是 ICLR（但我也不希望有哪个拖到 ICML 才刚写完），也许只是上次运气不好，ICLR 给我留下的印象尤其 toxic...

Jing 那边要在 graph tool 的基础上改，我看到这段话就已经开始两腿发软了，她说她有个在用这玩意的学生，我能不能求求她把我和那位大哥拉个合作，我可以写证明但我真的不想碰这个（（做梦（

Graph-tool is an efficient Python module for manipulation and statistical analysis of graphs (a.k.a. networks). Contrary to most other Python modules with similar functionality, the core data structures and algorithms are implemented in C++, making extensive use of template metaprogramming, based heavily on the Boost Graph Library. This confers it a level of performance that is comparable (both in memory usage and computation time) to that of a pure C/C++ library.
