定好了回北京的日期，暑假就这么匆匆忙忙地快要过完了。要说没怎么学也不完全是，对 network 确实比较完整地入了个门，也有信心秋季直接找海外导，只是我还不知道怎么接受“如果找不到应该怎么办”这样的结果。其实我早也有想到，在我导这边的话他本人已经 Associate Prof 所以没什么压力，何况我是要出国，搞数学也没得打杂，不管怎么 push 大概毕业之前我也发不出论文来，对我不上心也是正常的。之前一直对延毕是一个很乐的状态，临近了就笑不出来了，时常有失群落伍的惶恐，何况我这个科研进度，就算当我降转也实在有点落后，你统的学科特点摆在这里，当即只能想办法凑推荐信，也是希望能有人给我一点方向上的指点，自己整活儿还是太逆天了点。

但无论怎么样，暑假走过了很多以前想去或者想再去一次的地方（但我恨武汉，真的），参加了不少活动，即使离出国还比较遥远，也和朋友认认真真做了一次见面和告别（暑假开头在南昌的时候我和我姐戏称：这是我们一生仅一次的超长见面了。现在想想也许真的是吧，日暮途远，人间何世？），确认了我妈的状态，即使牺牲了很多搞学术的时间，也和她说了很多话，现在我已经没有什么好害怕的了。精神上来说，真的已经非常满足了。前两天有群友看完日记说我：

您这最近还这么划啊

就 我指的是 各处游览（）

我回复：那咋了

所以说现在甚至都有点变得很平常的一个心态了，我不知道未来是什么样子，我其实也觉得很恐惧，但它就是要来了。

8月19日

四刷 SJTU，也是第一次在夏天刷（，去面基了群友 明明是高中音游群的学弟为什么要叫群友，哥们弃明投暗降转了数学或许也有我的责任（悲。讨论了一些“为什么在校园里把一张交大地图铺在地上，会导致地图和地面有且仅有一相同点重合”（基于比例尺的压缩变换，考虑不动点。二维点直接算坐标线性变换也可）之类的问题，获得了一些数分高代的补习建议（，这就 GTM 135 启动（下午看了一下感觉是一本不强调有限维的线性代数，四舍五入还看了个泛函）。说真的是第一次觉得有人一起聊着天学数学是这么有趣的事情，以前没体验过，之后也不知道能不能再有机会了。

买到了玉兰苑的菠萝包，又幸福了，然而骑车去地铁站的路上摔了一跤压坏了一个有馅儿的，含泪吃掉。之前一直觉得玉兰面包房的奶黄可颂很好吃，馅料很实，可惜今天没买到，老板说是还没开学所以品种不全。下次还来。

今天第一次喝 Manner，瑞幸的确没得比，但是它真的好贵（，早说我自己带个杯子来（x

8月20日

你交这个蚊子真毒啊，昨天叮的包今天还又痒又痛，还有一个正好叮在手腕上，肿好大一片，不对称了（

连着好长时间没能睡个真正意义上的好觉了，今天睡了一下午，晚上还一直处在一个搞不清状况的状态（。

8月21日

最近经常感觉脖子像被掐了一样痛，尤其是只有几个点位很痛，可以轻松地用手指对准这几个点做出掐脖子的动作（，附带吞咽的时候喉咙痛，以及经常觉得大脑缺氧晕晕乎乎。虽然大概率就是坐姿问题导致颈椎出了点小毛病，小概率是睡多了，但还是挺怪的。

导今天，时隔 19 天，突然想起我来了（，只不过为什么想起我的方式是今天突然扔了两篇文章给我然后说明天可以聊一聊（，是要让我见到周四的日出吗，大致读一下算了。看了看他说的去浅搜了一下，其实我首先怀疑他这话不靠谱，至少之前一直观察到先做 cluster 再 estimation 这件事的远不如先 estimation 再 cluster 的多，我以前读 low-rank 那篇的时候有想过对 signed network 先 cluster 再做 completion，虽然二者看起来只是目标不同，但确实不清楚效率如何。我的理解是更大的困难在于不知道要怎么量化 k-means 分块的结果，在此基础上做 estimation 听起来有些太超越了，所以第一步至少不能做非常精细的 cluster。如果是先估计再 cluster 然后算 recovery rate 的话，只要比较两个矩阵就可以了，成熟的工具很多，Davis-Kahn 启动！找到一篇，怎么又 Ji Zhu了，这个领域真只有您一个组在做吗（（x

今天突然发现 Cytus II 确实挺好玩的，虽然读谱不能。这个就是说啊，学得越多，摸得越多，摸得越多，学得越少，所以学得越多，学得越少（

8月22日

这是一个什么情况啊（，奋战五个星期冲个 ICLR 吗，我大作业写得都没这么快（。说现在有个 idea 可以试试、做出来考虑投 ICML 的话，我觉得可以理解，但导今天在向我细细盘算如果十二月底 ICLR 被拒了转投 ICML 可能会来不及，他不会是认真的吧（（

还好昨晚没仔细看 Sofia 那两篇，而是出于一种莫名其妙的信赖去读 Ji Zhu 了，真是这个最有用，不然今天 simulation 都做不了。别看我平时这个啥 b 样子其实真有点活干的话不会这么闲的最怕的是在我还不知道要做什么的时候就真的没事情做（挠头

比较有意思的是它这个对于 non-linear smooth function 效果还是不错的，几乎可以说是非常漂亮的估计，但 linear 完全不行，看着跟重排了一样，调了很久 bandwidth 也没看出来有什么突破口，改成 polynomial 或者 sqrt 会稍微变好一点，但也经常发疯。这是一个什么原理，给我最直观的感受是这下 Lipschitz 函数族寄了（，那应该选一些什么东西，磨光函数吗，得加点什么条件才能更 general 捏。或者说 linear 寄了背后是和 uniform 有一定关系吗，搞了个 step function 也还是比 linear 好，这下 smoothness 大约也不是关键点了。

8月23日

昨晚摸了一个 step function，今天验出了一个对 step function 特化的优化，感觉听起来很有道理就是不知道具体什么道理（，但导看起来很有把握，好的（。往 sparse graph 上面套一套，无论什么方法都会寄，怎么会是呢（说明先别惦记你那 Erdos-Renyi 了 说明不需要在 real network 上做验证，代码就写到这儿了，已经不用继续战斗了（（x

然后浅看了一篇前两天没读的文章，我感觉我可能知道为什么对 step function 可以特化着做，以及为什么 linear 不行了，的确不是 smoothness 的问题（废话呢，有点意思的。

8月24日

摆烂一天，怎么混过明日组会，急在线等（

赶组会看论文见缝插针看了逃殿最新一话，感觉是截至目前质量最差单集（，被伤害到了（（

8月25日

今天漫无目的地一顿 search，哟怎么又是 Sischka，春季就看过他写的的一个 tutorial，他这篇这个 spline 我当初还寻思着什么玩意怎么会做这种东西，今天仔细看了一下，就一个 EM algorithm，没有具体的证明。

还没回学校就已经开始失眠了（，昨晚没睡好，下午收拾行李，一边收拾一边盯微信，盯了一下午没等到组会。上床躺了 10 min，醒来看到五分钟前开始了腾讯会议，导还给我打了个电话我没听到（。晚上吃完饭以为终于能睡会了，睡了一个半小时醒来发现导在群里 at 了我两次，他后来的话实在让我很怀疑，不会是以为我完全没理解他们在干什么所以不回的吧，真不是，但我又不好意思说我睡着了，好崩溃（（

但他画的饼是真的能吃饱（，赶 simulation 去了（（

8月26日

美好的一天从 simulation 又爆了开始，到 simulation 爆了好多次但找不出原因结束（，不知道一边听 兰若度母 一边写到底能起到一个什么样的效果（

今晚摆了，把书展买的涩泽龙彦的《虚舟》读了。买的时候拆开翻了一下感觉可以当睡前小故事一天一篇，一周就看完了，今天一读发现是重口睡前鬼故事（，喜欢，读完了，正好不用带去北京了（。

8月27日

之前精心设计了两个正好避免重复计数的算法，又精心设计了一些奇技淫巧使它们能跑得很快，还用小数据测了很多遍保证是对的，但就是估不准，测了一堆函数测得我怀疑这已经符合测不准原理了（，结果今晚发现 bug 出在这个 model 就是要重复计数，好幽默啊！但至少我 simulation 不爆了，数据量调大点的话效果挺好的，可喜可贺，学长牛逼。

回北京了，盒马的水果种类也变少了，无锡水蜜桃也吃不到了，还有另一个室友也在寝室里，想回家了（。从寒假开始我每次一过虹桥的检票闸机就掉眼泪，寒假的时候从上海一路哭到苏州才停（，今天车还没开就停了，可能是旁边有个姐一直在和家人打电话，说留学一年 150w 预算总能够用了，也不是拿不出来，把家里工作室那层楼卖了换个小点的就行了，听得我伤感不起来了（。我真是想通了，大家（？）延毕的理由不会都很正当吧，不会只有我是因为不愿意离开家所以想延毕先逃避个一年吧！（当然不可能完全是这个理由啊我开玩笑的！

8月28日

今早睡了十多个小时，晚上突然发觉好像最近忘记我半个月之后还有一场托福要考了！难怪这么悠闲（

8月30日

睡前看了一本在上海书展的时候挑中但没买的《在小山和小山之间》，豆瓣评分 8.2，我寻思着这不是稳了吗？但是当时在店里问店员能不能拆一本试读的时候被拒绝了，就感觉不放心，这都是线下书展了，好书会怕试读吗，会担心这一本的损耗吗，遂放回去没买，拿了无料就走了。在广西师大出版社那边三折的低价书都可以拆开试读，店员还掏出自己的钥匙帮我撕《虚舟》的包装，拆开觉得值得买也就买了。中国书籍出版社的摊位门可罗雀，店员是一位老先生，我站在他们店里读《被遗忘的王国：丽江 1941 - 1949》，后来一位老奶奶也想读这本，他还想再拆一本，我当时已决定要买，把样书放回去，才没有再拆，付了钱之后他还帮我盖好章放好藏书票一路夸我有眼光把我送出门（。

都到书展了，不读是真不能放心地买，读了觉得满意的我还大部分都没买（主要是拿不动，想着回来找个电子书看看再说。昨晚找了之后拿着平板躺上床，怎么只有 88 页，一口气读完了，结果气得我半夜睡不着。看了标题不看内容浑身难受，看的时候也浑身难受，看完了更是难受，还是三种不一样的难受，真有你的。东亚母女关系当然值得写，但决计不是这样写，看着像纪实文学，很多地方的确揪心，感同身受，但又觉得透着虚伪，结果作者在后记里终于坦白确实是编的，我服了。骗我感情就算了，好歹没骗到我的钱，这本打完折比《被遗忘的王国》还贵，唯一的亮点可能就是装帧和插画，真的不值。

没吃晚饭，夜宵去吃烤冷面，又双叒叕遇到了 neruko，neruko 好，烤冷面香香。
